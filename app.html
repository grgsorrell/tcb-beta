How to Sync Your Calendar

Step 1: Export Your Events
Click "Export .ics" in Settings above. This downloads a file called campaign-calendar.ics containing all your campaign events, deadlines, and tasks.
📅 Google Calendar
On Desktop:
1. Go to calendar.google.com
2. Click the + next to "Other calendars" on the left sidebar
3. Select "Import"
4. Choose the campaign-calendar.ics file you downloaded
5. Pick which calendar to add it to, then click "Import"
On Phone (Google Calendar app):
1. Open the .ics file from your Downloads
2. It should prompt you to add events to Google Calendar
3. Tap "Add all" or "Import"
🍎 Apple Calendar (iPhone / Mac)
On iPhone:
1. Tap the downloaded campaign-calendar.ics file
2. iOS will ask "Add All Events?"
3. Tap "Add All" — done!
On Mac:
1. Double-click the .ics file in Finder
2. Apple Calendar opens automatically
3. Click "Add All"
📨 Outlook
1. Open Outlook and go to Calendar
2. Click File > Open & Export > Import/Export
3. Choose "Import an iCalendar (.ics) file"
4. Select the downloaded file and click "Import"
Tip: Re-export anytime you add new events in the Toolbox. The .ics file always includes your full, up-to-date calendar.
📤 Importing Events INTO the Toolbox
Have events in another calendar? Export them as a .ics file from that app, then click "Import" in Settings above. We'll add all the events to your campaign calendar automatically.
